# Como rodar

1. Tenha o Node.js instalado (uma versão recente).
2. No terminal, vá até a pasta do projeto:

```bash
cd frontend
```

3. Instale as dependências:

```bash
npm install
```

4. Suba o servidor de desenvolvimento:

```bash
npm run dev
```

5. Abra o link que aparecer no terminal (geralmente http://localhost:5173).

Se der algum erro estranho, tenta remover a pasta `node_modules` e rodar `npm install` de novo.V
